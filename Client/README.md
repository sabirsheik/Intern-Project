# Intern Portal Frontend

## Setup

1. Install dependencies:
   npm install
2. Create `.env` from `.env.example`.
3. Start dev server:
   npm run dev

## Features

- Enterprise login page with demo credentials
- Role-aware dashboard for superadmin, admin, and intern
- Intern management, internship creation, assignment flow
- Report submission and listing
- Framer Motion animations and responsive Tailwind UI
